Data (자료)
프로그램이 처리할 수 있는 모든 것

Data type(자료형) : Data를 크기로 분류해 놓은 것 +언어의 종류에 따라 데이터 타입 구분이 달라지기도 함.
    -숫자형(Number)
    -문자형(String)
    -진리형(Boolean)
    -정의되지 않음(Undefined)
    -없음(Null)

컴퓨터의 단위
    비트(bit) - 2진수 1자리의 값을 저장할 수 있는 공간
    ->컴퓨터의 최소 단위.
    바이트(byte) - 8개의 비트로 구성된 단위
    ->데이터의 최소 단위(모든 데이터를 바이트 단위로 쪼개서 관리함)
    64비트 운영체제 - cpu가 한번에 계산할 수 있는 양이 64비트임. 만약 64비트를 넘는 
    주소를 부여받은 데이터가 있다면, 그 주소를 처리하기까지 2회의 계산을 해야함. 
    이러한 점을 방지하기 위해서, 주소는 컴퓨터가 한번에 처리할 수 있는 64비트를 사용함.


    데이터의 주소가 길어질 경우, 여러개의 바이트를 붙여서 사용할 수 도 있다.
    그런데, 몇개의 바이트를 연결했는지 알 수 가 없는 문제가 발생할 수도 있다. 
    이러한 문제를 해결할 수 있는게 바로 데이터 타입이다. 


->변수를 만들 때 사용!
변수란? 데이터를 저장하는 공간에 이름을 붙인것.
이름을 붙이는 이유 -> 저장한 데이터를 불러오는데 편하게 사용하기 위해서.(사람을 위해)

변수 사용 용도
1. 사용자의 입력값 저장
2. 계산 결과를 저장
3. 제어해야 하는 요소의 저장
4. 제어 조건의 저장
