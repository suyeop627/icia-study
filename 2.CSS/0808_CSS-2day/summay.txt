1.색상 처리(배경색, 글자색, 테두리 색 등)
    1) RGB(red,green, blue)함수 사용.
        각 색상별 0~255 사이의 수치를 조합.
        rgba(red, green, blue, alpha)함수 - 투명도 반영
        alpha : 투명도(0.0~1.0) 0에 가까울수록 투명해짐
        
   
    2) HEX : 16진수로 색생을 지정
        예) #FF00FF : 보라색
            #abc -> #aabbcc의 단축 표현./
             두자리씩 구분했을 때, 반복되는 두 수치가 있을 떄만.
            #aab3cc -> #ab3c  - 불가능
            투명도 처리 : 8자리 수치로 표현. 00 : 0.0, ff : 1.0            
  
    3) HSL : hsl(hue, saturation, lightness)함수 사용
        H - 색조, S - 채도 L - 명도
        H : 0~360까지의 색상환 각도로 표현
        S : 0~100%의 백분률 값. 0 - 회색음영, 100 - 색조의 본색
        L : 0~100%의 백분률 값. 0 - 검은색, 100 - 흰색
        예) red - hsl(0, 100%, 50%)
        투명도 반영 : hsla 함수 사용.

    요소의 크기 (너비와 높이) - 박스 모델(Box Model)
        문서를 구성하는 모든 요소는 box-model을 사용하여, 크기와 여백을 처리.

        일반적인 요소의 너비 = width +(2*padding) +(2*border)
        예) width: 100px;
            padding: 10px;
            -> 너비 = 120px.   <-padding과 border는 일반적으로 양쪽에 10px씩 적용되어, 
                                총 너비는 120px
            width:100px, padding: 10px, border:1px = 122px  
        일반적인 요소의 높이 = height + (2*padding) +(2*border)
            바깥족 여백(margin)은 요소의 크기에 고려하지 않는다.

        box-sizing 속성
        1) content-box : 기본값. 일반적인 크기로 처리
            실제 너비/높이 = 너비/높이 + 패딩 +테두리
        2) border-Box  : 길이(width/hight)에 패딩과 테두리가 포함
            실제 너비/높이 = 너비/높이
여백
    1) 안쪽 여백 (padding) : 테두리의 안쪽, 요소의 컨텐츠 바깥쪽
    2) 바깥쪽 여백(margin) : 테두리의 바깥쪽 영역 (요소 간의 간격 조정할 때 유용)

    padding 
        1)padding - 모든 여백
        2)padding-top   윗쪽 여백
        3)padding-right   오른쪽 여백
        4)padding-bottom    아랫쪽 여백
        5)padding-left  왼쪽 여백
        padding 속성 하나로 4방향을 단축하여 사용 가능.
        top을 시작으로 시계방향으로 나열 : top, right, bottom, left
        4개의 값을 지정 : top - right - bottom - left
        3개의 값을 지정 : top - right/left - bottom
        2개의 값을 지정 : top/bottom - right/left

        위/아래에 같은 값을 처리하고, 좌/우를 다른 값으로 처리하는 방법은 없음. -> 4개의 값을 지정해서 처리
        
        3방향은 같은 여백, 한 방향만 다른 여백을 줘야하는 경우
        -> padding: 10px; padding-left: 30px;
        전체여백 > 부분 여백 순으로 작성 - 요소와 가까운 순서로 우선순위가 부여되므로, 
                                         부분여백 먼저 설정한 다음에 전체 여백 주면, 부분여백 적용 안됨.

    margin  
        margin auto : 요소의 가운데 정렬에 사용 (margin의 활용은 padding과 동일하지만, auto값 설정은 margin에만 있는 요소 / 상하는 적용x 좌우만)
        예) margin: 0 auto ; +)값이 '0'일때는 단위 굳이 안적어도 됨     
    
    CSS의 단위
    1) px : 픽셀 단위. 1px = 1인치의 1/96
    2) pt : 포인트 단위. 1pt = 1인치의 1/76
    3) cm : 센티미터.
    4) em : 출판에서 사용하는 문자의 가로길이 (M 기준)
    5) 고정값 : thin(앏은,1~2px 정도), medium(중간, 3px정도), thick(두꺼운, 5~6px정도)
            -> 테두리에 사용
    6)% : 백분률. 


테두리(Border)
    요소의 테두리 선을 표현하는 속성.
    여백과 마찬가지로 4방향에 따로 테두리를 설정할 수 있음.
        1)border-style : 테두리의 종류(선의 모양). 실선, 점선, 파선 등
        2)border-width : 테두리의 두께.
        3)border-color : 테두리의 색상.
        4)border-radius : 모서리의 반지름(둥근 모서리 처리) - 좌측 상단이 top, 시계방향으로 진행
        5)border : 위의 각 속성을 합쳐서 사용하는 단축 속성. (단축 속성으로 radius는 설정 못함)
            두께 - 종류 - 색상 순서로 작성.
            (최소 두께와 종류는 작성해야 테두리가 나타난다.)
        +)border 색상의 기본값은 글자색(color)와 같음. 

minimum/maximum     width/height
    1) min-width : 지정된 범위 이하로 줄지 않는 요소.    
    2) max-width : 지정된 범위 이상으로 커지지 않는 요소로 설정

배경(background)
    요소의 배경 효과를 주기위한 스타일 설정 속성
    1)background-color : 배경 색상을 지정하는 속성.
    2)background-image : 배경 이미지를 지정하는 속성
    3)background repeat : 배경 이미지의 반복 여부 지정 속성
        repeat - x, y 반복 (기본값)
        no-repeat  반복없음
        repeat-x- 가로 방향으로만 반복    *repeat 활용해서 gradient 적용 가능
        repeat-y 세로 방향으로만로만
    4)background-size : 배경 이미지의 크기 지정 속성
        주요 속성 값
        1> 수치(px, % 등)
        2> contain : 이미지가 요소 영역 내부에 모두 출력
                     공백이 발생할 수 있음
        3> cover : 이미지가 요소 영역을 덮음
                  너비 공백은 발생하지 않으나, 높이 공백은 발생 할 수 있음.(너비기준으로 맞춤)
    5)background-attachment: 배경 이미지의 스크롤 여부 결정
        속성 값 : fixed(고정, 스크롤되지 않음)
                  scroll(스크롤 됨. 기본값)
    6)background-position : 배경 이미지의 위치 결정
        9곳의 위치를 지정
        ●left : 왼쪽 세로 중앙              left top/top left       top           right top / top right  
        ●right : 오른쪽 세로 중앙                
        ●top : 위쪽 가로 중앙                    left               center          right
        ●bottom : 아래쪽 가로 중앙
        ●center : 정중앙               left bottom / bottom left     bottom    right bottom/ bottom right
        ●left top : 왼쪽 위
        ●right top : 오른쪽 위
        ●left bottom : 왼쪽 아래
        ●right bottom : 오른쪽 아래
    7)background : 위 개별 속성을 하나로 처리하는 단축 속성
        크기(size)는 제외.
        순서 : 색상 - 이미지 - 반복여부 - 스크롤여부 - 위치

텍스트 관련 속성
    컨텐츠(내용)에 들어가는 글자(하위요소 포함)에 대한 설정
    ● color : 글자색
    ● text-align : 정렬 방식. left(기본값), center, right  (<div>태그 안의 <div>태그 있으면 text-align으로 정렬 가능)
                                justify(양쪽 정렬).
    ● vertical-align : 수직 정렬 방식. 이미지 + 텍스트인 경우 (보통 이미지가 더 크므로, 이미지 기준 상단, 중단 하단 위치 지정 )
        top : 이미지와 글자를 윗선에 정렬
        middle : 가운데(중앙선)에 정렬
        bottom : 밑선에 정렬(기본값)
    ● text-decoration : 글자 장식(선) 설정 및 제거
            overline - 윗줄
            underline - 밑줄
            line-through - 취소선(삭제선), 가운데 줄
            none - 줄 제거.(<링크관련 요소의 밑줄 제거)
    ● direction : 글자 인쇄 방향 설정, unicode-bidi와 함께 사용.
            ltr : 왼쪽에서 오른쪽으로 인쇄(기본값)
            rtl : 오른쪽에서 왼쪽으로 인쇄
            unicode-bidi속성 : 유니코드의 방향 설정 보조 속성.
    ● text - transform : 글자 변환. (알파벳기준)
        uppercase : 소->대문자로 변환
        lowercase : 대 ->소문자로 변환
        capitalize : 단어의 첫글자를 대문자로 변환
    ● text-indent : 들여쓰기. 픽셀 단위로 들여쓰는 너비 지정
    ● letter-spacing : 글자 간격
    ● word-spacing : 단어 간격    
    ● text-shadow : 글자에 그림자 효과를 추가
        설정 값 : 수평이동수치- 수직이동수치- 흐림수치- 색상

         ● white-space : 공백 문자에 대한 처리
            normal - 일반 상태 (기본값)
            nowrap - 연속 공백을 하나로 합침. <br>로 줄바꿈
            pre - 연속 공백 유지 (편집화면대로) <br>로 줄바꿈
            pre-wrap - 연속 공백 유지. 개행문자(엔터)와 <br>로 줄바꿈   +개행 : 줄바꿈
            pre-line - 연속 공백을 하나로 합침
                    개행문자와 <br>로 줄바꿈
    ● font-size : 글자의 크기를 지정
            사용단위 - px, em
            +)안전폰트 - 어느 환경에서건 항상 보일 수 있는 폰트
